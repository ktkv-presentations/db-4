Реляционная алгебра

Реляционная алгебра

формальная система для описания операций над отношениями (таблицами), которая используется для манипуляции и управления данными в реляционных базах данных

Операции реляционной алгебры как способ манипулирования данными

операции реляционной алгебры позволяют

  • Извлечение данных (выборка, проекция)
  • Объединение данных из разных таблиц (соединение, декартово произведение)
  • Модификация данных (пересечение, разность, объединение)

Основные операции реляционной алгебры

Объединение (Union)

позволяет объединить два отношения (таблицы), которые имеют одинаковую структуру (то есть, одинаковый набор и порядок атрибутов)

Условие выполнение

оба отношения должны иметь одинаковый набор атрибутов, иначе объединение невозможно


Таблица A: {1, 2, 3}
Таблица B: {3, 4, 5}
Результат объединения (A ∪ B): {1, 2, 3, 4, 5}

Основные операции реляционной алгебры

Пересечение (Intersection)

Операция пересечения возвращает строки, которые содержатся в обоих отношениях. Только те строки, которые существуют в обоих наборах данных, попадут в результат


Таблица A: {1, 2, 3}
Таблица B: {2, 3, 4}
Результат пересечения (A ∩ B): {2, 3}

Основные операции реляционной алгебры

Разность (Difference)

Операция разности возвращает строки, которые находятся в первом отношении, но отсутствуют во втором


Таблица A: {1, 2, 3}
Таблица B: {2, 3, 4}
Результат разности (A - B): {1}
Результат разности (B - A): {4}

Основные операции реляционной алгебры

Проекция (Projection)

используется для того, чтобы выбрать определенные столбцы (атрибуты) из отношения, исключая остальные


Таблица "Студенты" содержит следующие атрибуты: {ID, Имя, Возраст, Курс}
Если нам нужно только имя и курс студентов, мы можем выполнить проекцию на эти атрибуты:

Результат: {Имя, Курс}

Основные операции реляционной алгебры

Выборка (Selection)

Операция выборки (селекции) позволяет выбрать только те строки из отношения, которые соответствуют заданному условию


Таблица "Студенты" содержит следующие атрибуты: {ID, Имя, Возраст, Курс}
Если нужно найти всех студентов старше 20 лет, можно применить выборку по условию "Возраст > 20":

Результат: Строки, где возраст студентов больше 20.
